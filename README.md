## 🚀 Installation Guide

### 📦 Prerequisites
- ✅ JDK 17+
- ✅ Apache NetBeans 13
- ✅ Apache Tomcat 10

### 🛠 Setup Steps
1. **Configure JDK**:  
   `Tools → Java Platforms → Add JDK 17`

2. **Configure Tomcat**:  
   `Tools → Servers → Add Tomcat 10`

3. **Run Project**:  
   - Open project in NetBeans  
   - Right-click project → `Run`

## 🌟 Key Features

### 🛒 Shopping Features
- Product browsing with filters/sorting  
- Cart management  
- Order processing  
- Voucher system  
- Product feedback

### 🛠️ Admin Features
- Product management  
- Staff management  
- Inventory control  
- Sales analytics  
- Chatbot monitoring

### 📊 Statistics Module
- Revenue reports  
- Product performance  
- Order tracking  
- Data export (Excel)
